## Welcome to Polkadart

Polkadart is a powerful and type-safe Dart/Flutter SDK that provides developers with everything needed to build decentralized applications on Polkadot, Substrate, and other compatible blockchain networks. With comprehensive APIs, automatic code generation, and Flutter-first design, it's the most developer-friendly way to integrate blockchain functionality into your Dart applications.

All Polkadart packages are available on [GitHub](https://github.com/leonardocustodio/polkadart) with a [permissive Apache 2.0 license](https://github.com/leonardocustodio/polkadart/blob/main/LICENSE).

Polkadart is a modular ecosystem where each package serves a specific purpose. You can use them independently or together based on your needs.

### Core Packages

| Package | Description | Version |
|---------|-------------|---------|
| [**polkadart**](https://pub.dev/packages/polkadart) | Main SDK for blockchain interaction - provides APIs for connecting to nodes, querying state, and sending transactions | [![pub](https://img.shields.io/pub/v/polkadart.svg)](https://pub.dev/packages/polkadart) |
| [**polkadart_keyring**](https://pub.dev/packages/polkadart_keyring) | Secure key management and account handling - create wallets, manage keypairs, and sign transactions | [![pub](https://img.shields.io/pub/v/polkadart_keyring.svg)](https://pub.dev/packages/polkadart_keyring) |
| [**polkadart_cli**](https://pub.dev/packages/polkadart_cli) | Command-line tool for generating typed APIs from chain metadata | [![pub](https://img.shields.io/pub/v/polkadart_cli.svg)](https://pub.dev/packages/polkadart_cli) |
