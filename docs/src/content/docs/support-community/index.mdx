### Getting Started with Contributing

1. Read our [Contributing Guide](https://github.com/leonardocustodio/polkadart/blob/main/CONTRIBUTING.md)
2. Check [open issues](https://github.com/leonardocustodio/polkadart/issues) for tasks
3. Join our [Matrix chat](https://matrix.to/#/#polkadart:matrix.org) to discuss ideas
4. Fork the repository and make your changes
5. Submit a pull request
